# python-practice
关于python的小任务
## 这个仓库是什么？
我花了一周时间，每天做一个Python小项目，练习基础能力

## 包含的内容
|天数|项目|功能|
|------|------|------|
|第一天|打招呼|输入名字，输出问候|
|第二天|猜数字|猜1-100之间的随机数|
|第三天|计算器|加减乘除|
|第四天|待办事项|增删查|
|第五天|Excel筛选|读取excel，筛选年龄大于20的数据|
|第六天|持久化待办|任务保存到文件，关闭不丢失原先存储的数据|

## 怎么运行
1. 确保你安装了Python
2. 运行 `Python dayX.py`
3. 第五天需要下载pandas和openpyxl库
